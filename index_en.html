HTML

HTML means Hypertext Markup Language hypertext markup language) and is used to text-based Structure the content of a web document. There will not be only content is marked in it, but also meta information, describing this content.

CSS

CSS is the abbreviation for “Cascading Style Sheets”. CSS is one Computer language for building and structuring websites (HTML or XML). This language contains code elements and is made up of together these “Cascading Style Sheets”, which also contain CSS files (. css) are called.

Java-Script

JavaScript is a scripting language based on the Client side works. With the help of JavaScript you can increase the functionality of websites by changing content or be generated. JavaScript is used to replace CSS and HTML to complete.
